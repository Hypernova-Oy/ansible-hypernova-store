# Ansible
Nginx / PHP-FPM 7.2, Magento2 or Wordpress, Redis server cache, MariaDB, Fail2ban, Postfix relay to Mailgun, Firewall, Letsencrypt, and Linux malware detect. Ubuntu 18.04

The main idea is using [Ubuntu 18 LTS (Bionic Beaver)](https://wiki.ubuntu.com/BionicBeaver/ReleaseNotes) packages without custom compiled applications. It allows keeping a server updated and secure uses standard tools.
